Summary
World Wide Web
World Wide Web is mainly composed by HTML documents. HTML is short for Hypertext Markup Language, which is the language browser used to parse web page.
The web components include client (your computer and browser), Internet and servers.

Fundamental HTML Elements
HTML Elements are used to indicate how browsers display the contents in a web page.
Most HTML elements composed with opening tag, closing tag and the content between them.
<b>text</b> bond 'text'
<em>text</em> make 'text' italic

Attributes of HTML
Some tags have additional parameters called attribute. Attributes provide additional information to browser.
For instance, link tag uses attribute herf to quote URL.

<a href="url">link</a> creates a hyperlink with the attribute herl = url, the name of the hyperlink is link.
<img src="url" alt="text"> displays the image with address src = url (it can also be a local file of your computer),
if the image couldn't be shown, the attribute value of alt would be displayed.
